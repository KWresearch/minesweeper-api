Minesweeper, an HTTP API

TL;DR — Like the game, but as an HTTP API. You can check out the source code and a basic command line client example.

Base URL: https://minesapi.herokuapp.com/game (not available 24/7)

Start a new game
GET /game?difficulty=beginner
Starts a new game. If difficulty is not specified, it starts in beginner mode (3x3).
Query parameterExpected TypePossible values
difficulty string beginner, intermediate, advanced
Response Body (201 Created)

"status": "Playing now",
"data": {
"game_id": 2df73da661,

Reveal a tile
GET /board/{game_id}/reveal?row=row&col=col
Reveal a tile at position row, col, zero indexed. You must use the game_id returned in the previous call.
Query parameter KeyExpected TypePossible values
row integer [0, max_board_size]
col integer [0, max_board_size]
Response Body (200 OK)
{
"status": "You revealed position 1, 1",
"data": {
"game_id": 2df73da661,
"board": {
"tiles": [
[
{"row":0,"col":0,"revealed":false,"display":"."},
{"row":0,"col":1,"revealed":false,"display":"."},
{"row":0,"col":2,"revealed":false,"display":"."}
],
[
{"row":1,"col":0,"revealed":false,"display":"."},
{"row":1,"col":1,"revealed":true,"display":"2"},
{"row":1,"col":2,"revealed":false,"display":"."}
],
[
{"row":2,"col":0,"revealed":false,"display":"."},
{"row":2,"col":1,"revealed":false,"display":"."},
{"row":2,"col":2,"revealed":false,"display":"."}
]
],
"rows": 3,
"cols": 3
}
}
Put/remove a flag
GET /board{game_id}/flag?row=row&col=col
Put or remove a flag in the given position. Similar to "reveal a tile".
Query parameter KeyExpected TypePossible values
row integer [0, max_board_size]
col integer [0, max_board_size]
Response Body
{
"status": "Flag put on 1, 2",
"data": {
"game_id": 2df73da661,
"board": {
