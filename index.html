Minesweeper, an API

Start a new game

REQUEST

GET /game

You can specifiy a difficulty: /game?difficulty=beginnner|medium|advance. If nothing beginner is taken.

RESPONSE


Reveal a tile 

GET /board/:game_id/reveal?row=:row&col=:col

Where :game_id is the game's id returned in the previous call. As you can imagine, :row and :col are the tile positions (zero indexed).

RESPONSE


Put/remove a flag 

GET /board/:game_id/flag?row=:row&col=:col

The same logic for the parameters as in the previous call.
